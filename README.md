# HostsUpdate
+ 一个下载指定开源项目`hosts`文件并替换的`Python`脚本

####使用方法
+ 下载项目，在终端定位到`HostsUpdate.py`所在目录，执行 `$ python HostsUpdate.py`，然后输入当前用户密码获取`root`权限更新`hosts`


####Hosts来源：
+ 更新的`hosts` 来源是： [racaljk / hosts](https://github.com/racaljk/hosts)
